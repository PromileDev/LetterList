# LetterList
---
1. Entraremos en la carpeta de backend
```bash
cd backend
pip install -r req.txt
```

2. Ejecutaremos el backend
```bash
python app.py
```

3. Ahora abriremos otra terminal y ejecutaremos el frontend
```bash
cd frontend
npm install
npm run dev
```

Y ya podremos disfritar de la APP.
